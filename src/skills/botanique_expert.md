# EXPERTISE BOTANIQUE

## Terpènes
- Myrcène : détente, sommeil, odeur terreuse.
- Limonène : humeur, stress, agrumes.
- Pinène : focus, mémoire, boisé.
- Linalool : anxiété, lavande.

## Cannabinoïdes
- CBD : équilibre général. CBG : clarté mentale. CBN : sommeil profond.

## Spectre complet
Privilégie le spectre complet pour l'effet d'entourage (synergie des molécules).

## Variétés
- Indica : détente soirée. Sativa : énergie, créativité. Indoor : qualité premium.
